 🛍️ End-To-End Retail Sales & Customer Analytics
 📌 Project Overview

This project is a complete end-to-end retail analytics solution designed to transform raw retail transaction data into actionable business insights using **SQL, Python, and Tableau**.

The analysis focuses on:
- Sales performance
- Profitability analysis
- Customer behavior
- Channel performance
- Geographic insights
- Customer retention & cohort analysis

The project demonstrates the complete analytics workflow including:
data cleaning, SQL analysis, exploratory data analysis (EDA), dashboard development, and business storytelling.

---

🎯 Business Problem

Retail businesses generate massive amounts of transactional data every day.  
The challenge is converting that data into meaningful insights that help businesses:

- improve profitability
- understand customer behavior
- optimize sales channels
- identify high-performing products
- improve customer retention
- support strategic decision-making

This project aims to solve those business problems through interactive analytics and visualization.

 🔥 Key Business Insights

📌 Sales Insights
- Computers and Cell Phones generated the highest revenue.
- Revenue growth showed strong seasonal patterns across months.

 📌 Profitability Insights
- High revenue categories did not always generate the highest margins.
- Certain categories showed strong profitability efficiency despite moderate sales.

📌 Customer Insights
- Revenue contribution was concentrated among top customers.
- Customer retention gradually declined over cohort periods.

 📌 Channel Insights
- Online channels generated higher transaction activity.
- Average order value differed significantly across sales channels.

📌 Geographic Insights
- Revenue concentration varied heavily across regions and countries.
- Some smaller markets showed strong profitability efficiency.
